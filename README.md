# 📚 Study Routine Organizer

Um site moderno para **organizar rotinas de estudo**, acompanhar matérias e controlar o tempo de estudo de forma simples e eficiente.

---

## ✨ Funcionalidades

### 📅 Calendário de Estudos
- Calendário interativo para visualizar os dias do mês.
- Permite adicionar matérias ou tarefas de estudo em dias específicos.
- As matérias aparecem como etiquetas ou blocos no calendário.

### 📖 Gerenciamento de Matérias
- Adicionar, editar ou remover matérias.
- Cada matéria pode ter uma cor personalizada.
- Facilita a identificação visual no calendário.

### ⏱️ Cronômetro de Estudo
- Cronômetro integrado para sessões de estudo.
- Botões de:
  - **Iniciar**
  - **Pausar**
  - **Resetar**
- Ideal para técnicas de produtividade como **Pomodoro**.

### 📂 Menu de Navegação
- Menu localizado no **canto superior esquerdo**.
- Permite navegar entre:
  - Início
  - Calendário
  - Matérias
  - Cronômetro
